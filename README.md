## Instalacion
### Clonar el repositorio
```bash
git clone https://github.com/SrtaLiv/Consultorio-Nutricionista-v2.git
```

### Instalar dependencias
Puedes usar `npm`, `yarn`, `pnpm`, `bun`, ejemplo usando `npm`:
```bash
npm install
```

### Ejecutar el servidor de desarrollo
```bash
npm run dev
```
